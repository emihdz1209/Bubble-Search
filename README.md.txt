Este código implementa el algoritmo de ordenamiento Bubble Sort para ordenar dos vectores de enteros. Primero, define una función `swap` que intercambia dos elementos adyacentes en un vector. Luego, la función `bubbleSort` aplica repetidamente el proceso de comparación e intercambio a todos los elementos del vector, desplazando los valores más grandes hacia el final de la lista y dejando los más pequeños al inicio. El programa principal utiliza esta función para ordenar un vector predeterminado de números y luego solicita al usuario que ingrese siete calificaciones, que también se ordenan y se muestran en pantalla. Finalmente, el programa imprime la calificación más baja y la más alta de la lista ingresada por el usuario.
